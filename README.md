# 自建Blog网站

用Django+Vue建的个人博客网站，地址www.qinjiantester.cn（已过期未续费）

## 功能特性

- 首页推荐
- 测试经验Blog分享
- 个人生活分享
- 用户个人主页

## 技术栈

- **前端** ：HTML, CSS, Vue, React
- **前端** ：Python-Django
- **数据库** ：Mysql
- **其他** ：Docker, Nginx

## 安装
### 克隆项目

```bash
git clone https://github.com/yourusername/your-weibo-project.git
cd your-weibo-project
```
### 安装依赖

```bash
npm install

```
### 启动

```bash
npm start
```

## 访问
```bash
打开浏览器，访问 http://IP:PORT
```
